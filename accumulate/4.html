# 表现与数据分离是MVC框架的概念

MVC框架即 Modal-View-Controller, 用一种业务逻辑、数据、界面显示分离的方法，在改变界面和交互时不需要重写业务逻辑。

# 分为三个部分：
  1. var 数据 = { }
  2. 对数据操作显示在界面上，加载
  3. 界面上的UI固定不变
# 反例：

在js中直接写

object.html("<div><p>数据</p></div>");
